# CodeStudio

## 副标题
QR & Barcode Workspace

## 定位
专业、轻量、离线优先的中文二维码与条码工作站

### 支持平台
- Windows

---

## 核心原则

### P1 极致稳定
**优先级最高：**  
稳定性 > 性能 > 功能数量

**要求：**
- 不崩溃
- 自动恢复
- 自动保存

### P2 离线优先
**所有功能：**  
本地完成

**包括：**
- 生成
- 解码
- 加密
- 批量处理

无需云服务。

### P3 企业可用
**目标场景：**
- 仓库
- 实验室
- 学校
- 工厂
- 办公室

而不是只做个人玩具。

---

## MVP功能

### 模块1：生成器

**支持类型：**
- QR Code
- Text
- URL
- Email
- Phone
- WiFi
- JSON
- Barcode

**Barcode支持：**
- Code128
- Code39
- EAN13
- EAN8
- UPC
- DataMatrix

**导出格式：**
- PNG
- SVG
- PDF

---

### 模块2：解码器

**支持方式：**

1. **图片识别**
   - 拖拽支持：jpg, png, webp, bmp

2. **摄像头**
   - USB Camera
   - Laptop Camera
   - 实时扫描

3. **扫码枪**
   - USB HID
   - 自动识别

4. **多码识别**
   - 一张图：QR × 20, Barcode × 50
   - 全部解析

---

### 模块3：批量中心

**导入格式：**
- Excel
- CSV

**示例数据：**
```
id    name
A001  Dell
A002  HP
```

**模板：**
```json
{
  "id": "{id}"
}
```

**生成结果：**
- A001.png
- A002.png

---

### 模块4：加密二维码
**算法：** AES256-GCM

**支持方式：**
- 密码加密
- 企业密钥

**流程：**
1. 输入内容
2. AES加密
3. 生成二维码
4. 扫码输入密码
5. 解密显示

---

### 模块5：扫码工作台

**核心功能**

**界面预览：**
```
扫描结果
─────────────────
A001
A002
A003

共3条
```

**支持功能：**
- 去重
- 搜索
- 导出
- 时间戳

---

### 模块6：历史记录

**存储方式：** SQLite

**记录类型：**
- 生成记录
- 扫描记录
- 批量任务

**支持功能：**
- 搜索
- 收藏
- 标签

---

## 差异化创新

### 智能规则引擎
**示例规则：**
- A* → 资产
- SN* → 设备
- ORD* → 订单

扫码后自动分类。

### 自动动作
**规则流程：**
1. 扫描
2. 匹配规则
3. 执行动作

**可用动作：**
- 播放声音
- 保存数据库
- 导出CSV
- 调用Webhook

### 数字签名二维码

**生成结构：**
```json
{
  "id": "A001",
  "signature": "xxxxx"
}
```

**扫码验证：** ✓ Verified

**应用场景：**
- 防伪
- 门票
- 证书
- 资产标签

基于 Public-key cryptography。

## 技术架构

### 桌面端
**推荐技术栈：**
- Electron
- React
- TypeScript

**选择原因：**
- Claude生成质量高
- 生态成熟
- 插件丰富

### UI框架
- shadcn/ui
- TailwindCSS
- Framer Motion

### 状态管理
- Zustand（不要 Redux）

### 数据库
- SQLite
- 推荐：better-sqlite3

### QR生成
- 推荐：node-qrcode

### Barcode生成
- 推荐：JsBarcode

### 解码
- 推荐：ZXing JS

### Excel处理
- 推荐：SheetJS

### 加密
- Web Crypto API
- AES-256-GCM

> 不要自己实现密码学。

## 项目目标

构建一个专业级、离线优先、跨平台的二维码与条码工作站，支持生成、解码、批量处理、AES加密、扫码枪与摄像头扫描、规则引擎自动化和企业级稳定性。UI参考 Linear + Raycast，代码必须采用 TypeScript、React、Electron、SQLite 架构，保证高内聚低耦合、模块化和可测试性。
